# 知ってる？高松地方検察庁 クイズ

○×で答える全10問のブラウザクイズです。検察庁・検察官・検察事務官・高松地方検察庁・香川県の5ジャンルから、各ジャンル2問ずつ順番に出題されます。

## 特徴
- 5ジャンル×2問をジャンルごとに案内→第1問/第2問の流れで出題
- 30秒タイマーとタイムアウト時の効果音
- 正誤とプレイヤー回答・解説を結果画面で確認可能
- 問題データは `question.csv` から生成した `questions.json` を利用

## セットアップ
1. リポジトリを取得し、プロジェクトルートに移動。
2. ローカルサーバーで `index.html` を開く（例: `python -m http.server` など）。
3. ブラウザで表示。

## ファイル構成（主要）
- `index.html` — 画面構造とメインメニュー
- `styles.css` — 見た目とレイアウト
- `script.js` — 出題ロジック、タイマー、効果音、結果集計
- `questions.json` — 出題データ（`question.csv` から生成）
- `question.csv` — 元となる問題データ

## ゲームの流れ
1. メニューで「始める」を押すと、各ジャンルの案内が表示される。
2. 「第1問」「第2問」と効果音を鳴らしながら同ジャンル2問を連続出題。
3. 全10問終了後、結果画面に正解数と各問の正誤ボタンを表示。
4. ボタンをクリックすると、プレイヤー回答・正解・解説が確認できる。

## データ更新
- `question.csv` を編集し、`questions.json` を再生成して反映。
  - `category, question, answer, explanation` の順でCSV記述。
  - ○は `○`/`〇`/`Ｏ`/`O`/`o`/`1`/`true`、×はそれ以外で判定。

## 注意点
- ローカルサーバー経由で開かないと `fetch("questions.json")` がブロックされる場合があります。
- 効果音ファイル（`question01.mp3`, `timelimit01.mp3` など）が同ディレクトリに必要です。
